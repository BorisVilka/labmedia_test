## Тестовое задание Labmedia

Приветствую проверяющих! Ниже кратко описал, что сделал и зачем.

### Что реализовано
1. Три экрана с `ViewModel`: храню состояния экранов, списков и таймера.
2. Слои `domain` и `data` для удобного масштабирования и разделения ответственности.
3. Имитация сети через Retrofit: если сеть недоступна — вопросы читаются из `assets/questions.json`.
4. DI на `Dagger2` для управления зависимостями.
5. `DataBinding` для связи верстки и кода.
6. Состояния экранов: `loading / content / error`.
7. Имитация получения данных о тесте на первом экране (не было строго в ТЗ, но полезно для целостности приложения).
8. В `MyApp` зафиксировал тёмную тему, так как в Figma представлена только она.
